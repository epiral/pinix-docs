You do not need to install anything. Create an account and you can start using Agents and Clips.

<Steps>

1. **Create an account**

   Open [pinixai.com](https://pinixai.com) and create an account. After signing up, you automatically receive free Credit, enough for several months of everyday use.

2. **Open the Console**

   After signing in, open the Console. This is where you manage Clips, chat with Agents, and browse the Marketplace.

3. **Add a Clip**

   Open **Marketplace**, find the Clip you need, and click **Add**.

   The Agent gains the new capability immediately. For example, after adding the HackerNews Clip, you can ask the Agent: "What is hot on HN today?"

4. **Start chatting**

   Open **Agent Chat** and tell the Agent what you want to do in natural language. The Agent automatically chooses the right Clip to complete the task.

</Steps>

<Aside type="tip">
  Cloud Agents are powered by DeepSeek, with each conversation costing less than ¥0.01.
</Aside>

## What You Can Do

Different Clips give the Agent different capabilities:

| Clip | What the Agent can do |
|------|-----------------------|
| HackerNews | Fetch tech trends and search posts |
| Google | Search the web |
| Twitter | Search tweets and view trends |
| Feishu Docs | Create and edit documents |
| Todo | Manage task lists |

There are 50+ Clips in the Marketplace, with more being added continuously.
