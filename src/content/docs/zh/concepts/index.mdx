## Pinix 是什么

Pinix 是一个 **Agent Application OS**——AI Agent 的应用操作系统。

AI 的瓶颈不是模型本身，而是使用模型的能力。大多数人不知道怎么给 AI 配工具、怎么拆解任务、怎么纠错。Pinix 通过 **Clip** 生态解决这个问题——安装一个 Clip，Agent 就多一个能力。

就像手机有 App Store 和操作系统来管理 App，Pinix 有 Registry/Marketplace 和 pinix daemon 来管理 Clip。

```
手机：    App Store  +  iOS/Android  +  App
Pinix：  Marketplace +  pinix daemon +  Clip
```

## 一句话串起来

> 用户通过 **Agent** 说出需求，Agent 从 **Hub** 找到合适的 **Clip** 来完成任务。Clip 运行在 **pinix daemon** 上。安装更多 Clip，Agent 就能做更多事情。
