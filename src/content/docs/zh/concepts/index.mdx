Pinix 让 AI Agent 能使用任何设备的任何能力。你不需要为每个 Agent 搭建复杂的工具链，只需安装 **Clip** —— 自包含的能力单元，任何 Agent 都可以通过 **Hub** 发现和调用它们。

理解以下五个概念，你就理解了 Pinix。

## 架构概览

```
User ──► Agent ──► Hub ──────► Clip (SDK)
                    │             ↑
                    │          Runtime (管理生命周期)
                    │
                    └────────► Clip (Edge)
                                 ↑
                              Provider 协议
                              (bb-browser, clip-dock, ...)
```

**Agent** 理解用户想做什么。**Hub** 将请求路由到正确的 **Clip**。Clip 执行实际工作 —— 无论是搜索 Twitter、管理待办事项，还是控制浏览器。

**SDK Clip** 是由 Runtime 管理的 TypeScript 包。**Edge Clip** 通过 Provider 协议直接绑定硬件或 OS API。
