[安装 Pinix](/zh/getting-started/installation/) 后，只需三步就能完成第一次 Clip 调用。

<Steps>

1. **启动 daemon**

   ```bash
   pinix start
   ```

   这会在 9000 端口启动 `pinixd`，并在后台自动启动 Chrome 和 bb-browser。

   你应该会看到：
   ```
   Pinix started on :9000 (PID 12345)
   ```

2. **登录**

   ```bash
   pinix login
   ```

   这会打开一个浏览器窗口，进行设备码授权。按照页面上的说明登录你的 Pinix 账号。

   <Aside type="tip">
     还没有账号？在 [pinixai.com/register](https://pinixai.com/register) 注册。你也可以使用 token：`pinix login --token pnx_...`（从 [pinixai.com/settings](https://pinixai.com/settings) 获取）。
   </Aside>

   登录后，pinixd 会自动重启并连接到 Cloud Hub。你的本地 Clips 将可以从云端访问。

3. **打开 Console**

   ```bash
   open http://localhost:9000
   ```

   Console 是用于管理 Clips、与 Agent 对话、浏览 Marketplace 的 Web UI。

</Steps>

## 试试看

启动后，你可以立刻调用基于浏览器的 Clips（bb-browser 会在启动时注册它的平台适配器）：

```bash
# See what Clips are available
pinix hub list

# Search HackerNews
pinix invoke hackernews top

# Search Google
pinix invoke google search "pinix ai agent"
```

## 常用命令

| 命令 | 描述 |
|---------|-------------|
| `pinix start` | 启动 daemon |
| `pinix stop` | 停止 daemon |
| `pinix hub list` | 列出已安装的 Clips |
| `pinix hub add @scope/name` | 安装一个 Clip |
| `pinix invoke <alias> <command>` | 调用 Clip 命令 |
| `pinix upgrade` | 升级 Pinix |
| `pinix whoami` | 检查登录状态 |
