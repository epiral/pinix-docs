不需要安装任何东西。注册一个账号，就可以开始用 Agent 和 Clip。

<Steps>

1. **注册账号**

   打开 [pinixai.com](https://pinixai.com)，注册一个账号。注册后自动获得免费 Credit，足够日常使用好几个月。

2. **进入 Console**

   登录后进入 Console。这是你管理 Clip、和 Agent 对话、浏览 Marketplace 的地方。

3. **添加 Clip**

   进入 **Marketplace**，找到你需要的 Clip，点击 **Add**。

   Agent 立刻获得新能力。比如添加 HackerNews Clip 后，你可以直接问 Agent："今天 HN 上有什么热门？"

4. **开始对话**

   进入 **Agent Chat**，用自然语言告诉 Agent 你想做什么。Agent 会自动选择合适的 Clip 来完成任务。

</Steps>

<Aside type="tip">
  云端 Agent 由 DeepSeek 驱动，每次对话成本不到 ¥0.01。
</Aside>

## 你可以做什么

装了不同的 Clip，Agent 就有不同的能力：

| Clip | Agent 能做什么 |
|------|---------------|
| HackerNews | 获取技术热点、搜索帖子 |
| Google | 搜索网页 |
| Twitter | 搜索推文、查看趋势 |
| 飞书文档 | 创建和编辑文档 |
| 待办事项 | 管理任务清单 |

Marketplace 里有 50+ 个 Clip，持续增长中。
