## What Is Pinix?

Pinix is an **Agent Application OS** — an application operating system for AI Agents.

The bottleneck for AI is not the model itself, but the ability to use the model. Most people do not know how to configure tools for AI, break down tasks, or correct mistakes. Pinix solves this through the **Clip** ecosystem: install a Clip, and your Agent gains a new capability.

Just as phones use an App Store and an operating system to manage Apps, Pinix uses the Registry/Marketplace and the pinix daemon to manage Clips.

```
手机：    App Store  +  iOS/Android  +  App
Pinix：  Marketplace +  pinix daemon +  Clip
```

## In One Sentence

> A user tells an **Agent** what they need, the Agent finds the right **Clip** from the **Hub** to complete the task, and the Clip runs on the **pinix daemon**. Install more Clips, and the Agent can do more.
